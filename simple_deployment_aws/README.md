# Simple WordPress Deployment on AWS

## Summary
A simple project to deploy and bring up a wordpress site on AWS. The wordpress will be deployed on a EC2 instance inside a VPC. 
The EC2 instance will be accessable from internet via the Internet Gateway. The MySQL DB for the wordpress will be provisioned in the private subnet of VPC


## Resources
* VPC
* Internet Gateway
* Security Groups
* EC2 
* Public Subnet
* Private Subnet
* RDS DB
